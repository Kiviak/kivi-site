# *kivi-site*  
a ebook management site.see [http://34.196.222.222/](http://34.196.222.222/)
## features:  
1. search
2. online reading
3. download
4. abstract of book
5. user can review and star books  
## powered by:
1. *Web framework*: Django
2. *Web Page*: jQuery & CSS
3. *Server*: Nginx & uWSGI
4. *Database*: MySQL
5. *Cache*: Redis
6. *Search*: Elasticsearch
